# Address Book System in C

## Project Overview

The **Address Book System** is a console-based application written in C that allows users to efficiently manage and organize contact information. The system provides functionalities for creating, displaying, updating, and deleting contacts in a structured and user-friendly way.

---

## Features

* **Add Contact**: Create new contacts with name, phone number, email, and address.
* **Display Contacts**: View all saved contacts in a neatly formatted table.
* **Search Contacts**: Quickly find a contact using name or other attributes.
* **Update Contact**: Modify existing contact details.
* **Delete Contact**: Remove unwanted contacts.
* **Sort Contacts**: Sort contacts based on user-defined criteria.

---

## Menu Interface

The system uses a simple **text-based menu** for interaction:

```
1. Create Contact
2. Display Contacts
3. Search Contact
4. Update Contact
5. Delete Contact
6. Exit
```
